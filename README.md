# itqu

voidfelt generator

![screenshot](https://raw.githubusercontent.com/eelfroth/voidscribble/master/screenshot.png)

## controls

`r R`  random

`m M`  mesh


`s  `  save png

`e  `  export svg


`+ -`  vertex count

`. ,`  stroke weight

## build

run with [Processing 3.x](https://processing.org/download/)
